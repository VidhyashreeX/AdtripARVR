# Assets Folder

Place your transparent PNG dress images here.

## Image Requirements

- **Format**: PNG with transparent background
- **Recommended Size**: 1024x1024px or 2048x2048px
- **Naming**: dress1.png, dress2.png, dress3.png, etc.

## Creating Dress Images

1. Find or create dress images
2. Remove background (use tools like remove.bg or Photoshop)
3. Save as PNG with transparency
4. Ensure the dress is centered and properly oriented

## Example Sources

- Freepik (free with attribution)
- Pngtree
- Flaticon
- Create your own using design tools

## Tips

- Use high-resolution images for better quality
- Keep file sizes reasonable (< 500KB) for faster loading
- Test different scales and positions in app.js for proper fitting
